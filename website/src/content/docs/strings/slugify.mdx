Converts a string to a URL-friendly slug by lowercasing, stripping accents, and replacing non-alphanumeric characters with a configurable separator (default `-`).

## Import

```ts
import { slugify } from "1o1-utils";
```

```ts
import { slugify } from "1o1-utils/slugify";
```

## Signature

```ts
function slugify({ str, separator }: SlugifyParams): string
```

## Parameters

| Name      | Type     | Required | Description                                             |
| --------- | -------- | -------- | ------------------------------------------------------- |
| str       | `string` | Yes      | The string to slugify                                   |
| separator | `string` | No       | The separator to use between words (default `-`)        |

## Returns

`string` — A lowercase slug with non-alphanumeric runs replaced by `separator`.

## Examples

```ts
slugify({ str: "Hello World!" });
// => "hello-world"

slugify({ str: "Café au lait" });
// => "cafe-au-lait"

slugify({ str: "  --Multiple   Spaces--  " });
// => "multiple-spaces"

slugify({ str: "Hello World!", separator: "_" });
// => "hello_world"

slugify({ str: "Hello World!", separator: "" });
// => "helloworld"
```

## Edge Cases

- Throws if `str` is not a string.
- Throws if `separator` is not a string.
- Handles Unicode normalization (NFD) to strip accents.
- Removes leading/trailing separators (the separator is regex-escaped, so any string is safe).
- Collapses multiple non-alphanumeric characters into a single separator.
- An empty `separator` skips trimming and concatenates the alphanumeric runs.
